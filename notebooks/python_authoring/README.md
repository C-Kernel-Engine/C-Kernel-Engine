# Python Authoring Notebooks

This folder groups the notebook lanes that sit on top of the existing Python authoring and API surface.

Current lanes:

- `v7_training/`
  Ordered v7 notebook lane for training demos, onboarding, dataset preparation, and artifact inspection.

Primary launch path:

```bash
.venv/bin/jupyter lab notebooks/python_authoring/v7_training/
```

Compatibility note:

- `notebooks/v7_training/` remains available as a compatibility alias so existing commands, demos, and links do not break while the notebook surface is being reorganized.
