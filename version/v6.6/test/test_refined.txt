  Issue #11: KV Cache Layout Mismatch ✅ VERIFIED                                                                                                                                                                                            
                                                                                                                                                                                                                                             
  Documented: v6.6 uses token-major layout [layer][k/v][seq_pos][kv_head][head_dim] but kernel expects head-major                                                                                                                            
                                                                                                                                                                                                                                             
  Evidence in ck-kernel-inference.c:911-912:                                                                                                                                                                                                 
  (float*)(model->kv_cache + 0 * 2 * MAX_SEQ_LEN * NUM_KV_HEADS * HEAD_DIM),  /* K cache */                                                                                                                                                  
  (float*)(model->kv_cache + (0 * 2 + 1) * MAX_SEQ_LEN * NUM_KV_HEADS * HEAD_DIM),  /* V cache */                                                                                                                                            
                                                                                                                                                                                                                                             
  This formula computes offset as layer * 2 * MAX_SEQ_LEN * ... which is token-major (seq_pos as the fast-varying dimension).                                                                                                                
                                                                                                                                                                                                                                             
  v6.5 uses head-major: [head][token][head_dim] with kv_head_stride = aligned_context_window * aligned_head_dim

  Root Cause Analysis                                                                                                                                                                                                                        
                                                                                                                                                                                                                                             
  Primary Root Cause: Codegen Doesn't Respect IR                                                                                                                                                                                             
                                                                                                                                                                                                                                             
  The fundamental design goal of v6.6 is:                                                                                                                                                                                                    
  Template → IR1 → Fusion → IR Lower → Codegen (pure translation)                                                                                                                                                                            
                                                                                                                                                                                                                                             
  But the actual implementation:                                                                                                                                                                                                             
  Template → IR1 → Fusion → IR Lower → Codegen (IGNORES IR, uses hardcoded values)                                                                                                                                                           
                                                                                                                                                                                                                                             
  Evidence in codegen_v6_6.py:                                                                                                                                                                                                               
                                                                                                                                                                                                                                             
  1. Line 502-506: Hardcoded token_offset = 16 default                                                                                                                                                                                       
  2. emit_op(): Uses static buffer calculations instead of IR offsets                                                                                                                                                                        
  3. No ops parameter reading: generate() reads IR but emit_decode_function() doesn't use the IR offsets                                                                                                                                     
                                                                                                                                                                                                                                             
  Secondary Issues (All Flow from Root Cause)

  Recommendations for Improvement                                                                                                                                                                                                            
                                                                                                                                                                                                                                             
  1. Fix Codegen to Be a Pure IR Translator                                                                                                                                                                                                  
                                                                                                                                                                                                                                             
  Current emit_op() approach:                                                                                                                                                                                                                
  def emit_op(op: Dict) -> str:                                                                                                                                                                                                              
      # Uses hardcoded logic per kernel type                                                                                                                                                                                                 
      if function == "embedding_forward_q8_0":                                                                                                                                                                                               
          tok = get_offset(activations, "tokens")  # May fail                                                                                                                                                                                
          # ...                                                                                                                                                                                                                              
                                                                                                                                                                                                                                             
  Recommended approach:                                                                                                                                                                                                                      
  def emit_op(op: Dict) -> str:                                                                                                                                                                                                              
      # Read EXACTLY from IR - no hardcoded fallbacks                                                                                                                                                                                        
      inputs = op.get("activations", {})                                                                                                                                                                                                     
      outputs = op.get("outputs", {})                                                                                                                                                                                                        
      scratch = op.get("scratch", [])                                                                                                                                                                                                        
                                                                                                                                                                                                                                             
      # Each input has activation_offset from IR Lower 2                                                                                                                                                                                     
      for name, info in inputs.items():                                                                                                                                                                                                      
          offset = info["activation_offset"]  # Trust the IR!                                                                                                                                                                                
                                                                                                                                                                                                                                             
  2. Add Pre-Codegen Validation                                                                                                                                                                                                              
                                                                                                                                                                                                                                             
  Add a validation pass between IR Lower 2 and Codegen:                                                                                                                                                                                      
  def validate_ir_for_codegen(ir_ops):                                                                                                                                                                                                       
      for op in ir_ops:                                                                                                                                                                                                                      
          assert "activation_offset" in op["activations"], f"Op {op['idx']} missing offsets"                                                                                                                                                 
          for scratch in op.get("scratch", []):                                                                                                                                                                                              
              assert scratch.get("scratch_offset") is not None, f"NULL scratch in op {op['idx']}"                                                                                                                                            
                                                                                                                                                                                                                                             
  3. Add Integration Test
