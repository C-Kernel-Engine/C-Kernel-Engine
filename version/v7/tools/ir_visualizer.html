Math Operator Math Intuition

Quick formula memory cards for operators seen in this IR. Cards are highlighted when matching ops are present in loaded decode/prefill data.

RoPE (Q/K Rotation)

not detected
[q'_{2i}, q'_{2i+1}] = R(θ_i) · [q_{2i}, q_{2i+1}] q'_{2i} = q_{2i} cos θ_i - q_{2i+1} sin θ_i q'_{2i+1} = q_{2i} sin θ_i + q_{2i+1} cos θ_i

Intuition: each 2D pair rotates by position-dependent angle; attention dot-products then carry relative position information.

GELU

not detected
GELU(x) = 0.5 x (1 + tanh(√(2/π) (x + 0.044715 x^3)))

Intuition: smooth gating. Large positive values pass through, negatives are softly suppressed.

SwiGLU

not detected
SwiGLU(xW, xV) = (xW) ⊙ swish(xV) swish(z) = z · σ(z), σ(z)=1/(1+e^{-z})

Intuition: one branch carries value, the other branch gates it with a smooth sigmoid-shaped control.

RMSNorm

not detected
RMSNorm(x) = γ ⊙ x / √(mean(x²) + ε)

Intuition: normalize magnitude, preserve direction; stabilizes residual stream before projections.

Workflows Common Debugging Workflows

New Runtime Issue

# 1. Find first failure
python3 test_layer_by_layer.py --model "$MODEL_DIR" --token 25

# 2. Debug at that layer
python3 v7_comprehensive_debug.py --model-dir "$MODEL_DIR" --token 25 --stop-at LAYER --threads 1 --context-len 1024

# 3. Trace divergence
python3 trace_divergence.py --model "$MODEL_DIR" --token 25 --threads 1 --context-len 1024

NaN Detected

# 1. Find layer with NaN
python3 trace_nan_layer.py --model "$MODEL_DIR" --token 25

# 2. Find exact op with NaN
python3 trace_nan_step.py --model "$MODEL_DIR" --token 25

# 3. Check weight integrity
python3 test_weight_offset_consistency.py --model-dir "$MODEL_DIR"

Memory Corruption

# 1. Quick memory check
python3 test_memory_planner.py --layout="$CK_BUILD_DIR/layout_decode.json"

# 2. Deep validation
python3 advanced_memory_validator.py --model-dir "$MODEL_DIR"

# 3. BUMP sync check
python3 test_bump_layout_sync.py --model-dir "$MODEL_DIR"

Converter Bug

# 1. Check weight offsets
python3 test_weight_offset_consistency.py --model-dir "$MODEL_DIR"

# 2. Verify dtype consistency
python3 test_dtype_consistency.py --model-dir "$MODEL_DIR"

# 3. GEMV specific test
python3 test_gemv_q8_0_q8_0.py
